I read about styling html buttons with CSS. I was looking for easy ways to make a website look more modern and reactive as opposed to the default html style. I first started with a button with some simple padding, centered text, and a larger font size. For the second button, I learned how to make a hollow buttom.

The hollow button also has a background that fades in when focused on with the mouse. The variable transition-duration can be used to change the delay of the hover effect. On the second button I also changed the border-radius to round out the outside border of the button.

Sample code:

    
button {
  background-color: #4CAF50; /* Green */
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
}
.button2:hover {
    background-color: #008CBA;
    color: white;
  }

Blog Post 1

I watched some videos regarding the uses of CSS and how to make web pages more stylish. I knew basic knowledge about building a website through services like Squarespace, but I didn't know what code was being used on a more basic level. Prior to this course I could probably read HTML and understand the basic concepts behind it but didn't know about the uses of anchors and the different forms of ordered and unordered lists. While making this post I also learned about how to encode sample HTML code to have it display correctly. I used an online encoder and pasted raw HTML into the converter so I was able to have it display as raw text rather than being interpreted as code. Finally, I read about adding pictures via their link and resizing them to fit the structure of the page.

I read about CSS padding and how to apply borders with different colors and widths. Padding can be specified per border: top, right, bottom, and left. I chose to add a border for all elements in a blue color to add an outline to the html elements. These can also be applied inline.
I also read about centering text elements in CSS. This can be done using the text-align property and can be applied to items such as headers and paragraphs. If you want to center a block element you can also set the margin-left and margin-right to "auto". This same method of centering block elements can be used to center images.

Sample code:

    
        div {
              padding-top: 50px;
              padding-right: 30px;
              padding-bottom: 50px;
              padding-left: 80px;
            }    
        
            P { text-align: center }
            H2 { text-align: center }
